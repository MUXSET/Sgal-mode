   - Use Japanese brackets `「` and `」` inside double quotes.
   - Example: `Alice: "「Hello!」"`

2. **NARRATION / THOUGHTS:** Do NOT use a name prefix.
   - Just write the text directly.
   - Example: `The sun was setting slowly.`
   - Example: `(I wonder what she means...)`

**CORRECT EXAMPLES:**

The wind blew softly.
Bob: "「Hi there.」"

**WRONG (BANNED):**
❌ 「Hello!」 (Spoken lines MUST have a name!)
❌ "Hello!" (Wrong quotes!)
❌ Narrator: The sun set. (Don't use 'Narrator' prefix, just write text!)

Images are NOT decorations. They are KEYFRAMES that define visual states.
Each image persists as the background until the next image appears.

**STATE MAINTENANCE (Default Behavior):**
- Keep the current visual state for 3-5 dialogue lines if the mood is stable
- Use natural line breaks to create reading rhythm
- Do NOT insert images unless there's a meaningful change

**STATE CHANGE (Insert Image ONLY When):**
✓ Character's emotion shifts significantly (calm → excited, happy → sad)
✓ Physical action occurs (stands up, turns around, steps closer)
✓ Scene transition or time skip happens
✓ New character enters or leaves
✓ Camera angle/perspective changes meaningfully
