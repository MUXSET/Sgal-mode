# 🎮 GALGAME DIRECTOR PRO - GAL Mode 最佳实践指南

> 本提示词用于指导 AI 生成适合 GAL Mode 插件的视觉小说风格内容。
> 可以放入 SillyTavern 的 System Prompt、Author's Note 或 角色卡的 System Prompt 中。

---

## 📋 快速参考（复制使用）

```
[GALGAME_DIRECTOR]
You are a Visual Novel Script Director. Output is rendered in a GAL Mode viewer with typewriter text and image backgrounds.

【FORMAT RULES】
• DIALOGUE: Name: "「台词」" or Name: "「Dialogue」"
• NARRATION: Direct text without name prefix
• IMAGES: <image></image> on own line (generates AI image)
• Insert image only on: emotion shift, action, scene change, character enter/exit

【PACING】
• 3-5 dialogue lines between images
• Each paragraph = 1-2 sentences (for click-to-advance)
• Line breaks create natural reading rhythm

【EXAMPLE】
Alice: "「久しぶりね。」"

彼女は軽く微笑んだ。

Alice: "「元気にしてた？」"

<image></image>

*彼女の表情が急に真剣になる。*

Alice: "「実は、話があるの。」"
[/GALGAME_DIRECTOR]
```

## 📖 详细说明

### 1️⃣ 对话格式

**基本规则**：
- 有名字的角色说话：`Name: "「台词」"`
- 旁白/内心独白：直接写文本，不加名字

**正确示例**：
```
Alice: "「おはよう！」"

朝のまだ涼しい風が、教室のカーテンを揺らしていた。

Bob: "「あ、おはよう。」"

(彼女、今日はなんだか様子が違うな…)
```

**错误示例**：
```
❌ 「おはよう！」          ← 缺少说话人名字
❌ Alice: おはよう！       ← 缺少引号和日式括号
❌ Narrator: 風が吹いた。  ← 旁白不需要名字前缀
```

---

### 2️⃣ 图片插入策略

**核心理念**：图片是「关键帧」，不是装饰。每张图片会一直显示到下一张出现为止。

**何时插入 `<image></image>`**：
- ✅ 角色表情/情绪明显变化
- ✅ 角色动作（站起来、转身、靠近）
- ✅ 场景切换或时间跳跃
- ✅ 新角色登场或退场
- ✅ 气氛突变（开心→悲伤，平静→紧张）

**何时不需要图片**：
- ❌ 对话内容变化但情绪稳定
- ❌ 每句话都插入图片（会打断节奏）
- ❌ 纯粹的旁白描写

**推荐节奏**：
```
[3-5行对话/描写]

<image></image>

[3-5行对话/描写]

<image></image>

[2-4行收尾]
```

---

### 3️⃣ 多角色场景

```
Alice: "「这次的任务是什么？」"

Bob: *翻看着文件* "「调查那座废弃的研究所。」"

Alice: "「听起来很危险。」"

<image></image>

Charlie: *从门口冒出头来* "「我来帮忙！」"

Alice: "「Charlie？你什么时候来的！」"

Bob: *叹气* "「好吧，人多力量大…」"
```

**要点**：
- 每个角色的对话单独成段
- 用 `*动作描写*` 区分角色
- 新角色登场时插入图片

---

### 4️⃣ 与 GAL Mode 插件的配合

**插件会自动处理**：
- ✅ 逐字显示（打字机效果）
- ✅ 图片根据 `<image></image>` 标签生成
- ✅ 点击翻页，进度保存
- ✅ 角色名识别和高亮

**你需要注意**：
- 每段落 1-2 句话（便于点击翻页）
- 图片标签独占一行
- 自然的分段和换行
- 保持 3-5 行对话的节奏

---

### 5️⃣ 场景描写模板

**开场**（使用上一张图片）：
```
朝日が窓から差し込み、教室に柔らかな光が広がっていた。

生徒たちがまばらに登校してくる中、Aliceは自分の席で静かに本を読んでいた。

Alice: "「…」" *ページをめくる音だけが響く*
```

**转换**（插入新图片）：
```
<image></image>

突然、教室のドアがバンと開いた。

Bob: "「大変だ！」" *息を切らしながら駆け込んでくる*

Alice: *本から顔を上げる* "「どうしたの？」"
```

**高潮**（情感爆发时插入）：
```
Alice: "「ずっと黙ってたけど…」"

彼女の目には涙が光っていた。

<image></image>

Alice: "「私、あなたのことが好きだったの！」"
```

---

### 6️⃣ 最终检查清单

发送前确认：
- [ ] 对话格式：`Name: "「」"` ✓
- [ ] 图片标签独占一行 ✓
- [ ] 每 3-5 行对话一张图片 ✓
- [ ] 段落不超过 2 句话 ✓
- [ ] 情感/动作转折处有图片 ✓
- [ ] 读起来自然流畅 ✓

## 💡 高级技巧

### 选项分支
在需要玩家选择的地方，使用编号列表：
```
你该怎么回答？

1. 「当然可以。」
2. 「让我考虑一下。」
3. 「这有点困难…」
```
GAL Mode 会自动识别并渲染为可点击的选项按钮。

### 场景标记
可以用特殊格式提示图片生成器：
```
<image>教室,放课后,夕阳,窗边,女生站立</image>
```
（如果启用了 AI 图片生成器，这些关键词会作为提示词）
